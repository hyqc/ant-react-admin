### 登录

###### POST /admin/base/login

###### 参数

| 字段     |  类型  | 是否必须 | 示例   | 备注       |
| -------- | :----: | :------: | :----- | :--------- |
| name     | string |    是    | admin  | 管理员名   |
| password | string |    是    | 123456 | 管理员密码 |
| vcode    | string |    是    | 1234   | 验证码     |

###### 返回

```json
{
  "code": 0, // 状态吗
  "message": "登录成功",
  "data": {
    "admin": {},
    "menus": []
  }
}
```

### 登出

###### POST /admin/base/logout

###### 参数

| 字段 |  类型  | 是否必须 | 示例  | 备注      |
| ---- | :----: | :------: | :---- | :-------- |
| id   | string |    是    | admin | 管理员 ID |

###### 返回

```json
{
  "code": 0, // 状态吗
  "message": "退出成功"
}
```
